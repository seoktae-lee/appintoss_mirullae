## 1. 서비스 개요

### 1.1 한 줄 정의
시험일과 목표 분량만 입력하면, 오늘 미룬 분량의 대가를 내일의 숫자로 잔인하게 재계산해 주는 10초 컷 벼락치기 플래너.

### 1.2 앱 이름
**미룰래** (영문: Mirullae)

### 1.3 핵심 정체성
**팩트 폭행 툴** — 실용 플래너가 아니라, 미루면 숫자로 혼나는 재미+바이럴 중심 앱.

### 1.4 최우선 가치
**팩트 폭행 임팩트** — 캐릭터 피폐 + 배경 어두워짐 + 큰 숫자 + 잔인한 문구, 이 4가지 연출에 가장 공을 들임.

### 1.5 해결하는 문제
- 시험/자격증 앞두고 거창한 플래너 쓰다가 계획 수립 자체에 지침
- "오늘 안 하면 내일 얼마나 해야 하는지" 체감이 안 됨
- 미루는 자신을 통제할 유머러스한 동기 부여가 없음

### 1.6 핵심 가치
1. **10초 설정** — 미션명 + 마감일 + 총분량 + 단위. 끝
2. **팩트 폭행** — 미루면 내일 할 양이 늘어나는 잔인한 숫자
3. **캐릭터 피폐** — 미루기 횟수에 따라 캐릭터가 점점 피폐해짐
4. **바이럴 공유** — 팩트 폭행 결과를 인증 카드로 공유

## 3. 핵심 기능 A — 목표 설정

### 3.1 입력 항목

| 항목 | 필수 | 입력 방식 |
|------|------|-----------|
| 미션명 | **필수** | 텍스트 (예: SQLD 자격증) |
| 마감일 | **필수** | 날짜 선택 (오늘 이후) |
| 총 분량 | **필수** | 숫자 입력 (예: 40) |
| 분량 단위 | **필수** | 선택: 페이지 / 강의 / 문제 / 회 / 개 |

### 3.2 입력 UI
- **바텀시트** 형태로 표시
- 목표 추가 버튼: **우측 상단 FAB** (+ 아이콘, TDS 스타일)

### 3.3 목표 개수
- 최대 **3개** 동시 관리 가능
- 3개 초과 시 "목표는 3개까지만 가능해!" 안내

### 3.4 자동 계산
- 남은 일수 = 마감일 - 오늘
- 오늘 할당량 = 남은 분량 ÷ 남은 일수 (소수점 첫째 자리)
- 하루 리셋 시점: **자정 (00:00)**

### 3.5 목표 수정/삭제
- TDS 컴포넌트 활용 (ConfirmDialog 등)
- 삭제 시 확인 다이얼로그

## 4. 핵심 기능 B — 매일 할당량 확인 + 선택

### 4.1 홈 화면 구성 (캐릭터 히어로 레이아웃)
- **원페이지** (탭 없음)
- 우측 상단: 설정 [⚙] + 목표 추가 [+] FAB
- 구조 (위→아래):

```
┌─────────────────────────┐
│                    [⚙][+]│
│                          │
│     (큰 캐릭터 히어로)     │
│     미루기 상태 반영       │
│                          │
│  "오늘도 파이팅!" (인사말)  │
│  목표 2개 · 미루기 0회     │
│                          │
│  ┌─ 미루기 지수 링 차트 ─┐ │
│  │  중앙: 점수/등급 표시  │ │
│  └───────────────────┘ │
│                          │
│  ● ○ (dot indicator)    │
│  ┌─ 목표 카드 (스와이프) ─┐│
│  │ SQLD 자격증     D-17  ││
│  │ 오늘: 2.4강의         ││
│  │ ████░░░ 58%          ││
│  │ [했어요][조금][미룰래]  ││
│  └───────────────────┘ │
│                          │
│  ┌─ 미루기 캘린더 ───────┐│
│  │ 월 화 수 목 금 토 일   ││
│  │ 😊 😊 😐 😨 😊 ·  ·  ││
│  │ (날짜별 캐릭터 표정)   ││
│  └───────────────────┘ │
└─────────────────────────┘
```

### 4.2 미루기 지수 (링 차트)
- 첫번째 레퍼런스(Intelly 헬스 스코어)처럼 **원형 링 차트**로 시각화
- 중앙에 미루기 지수 점수 또는 등급 표시
- 링의 색상/채움 정도가 미루기 상태를 반영
- 미루기 0회: 초록 100% → 미루기 많을수록 빨간색으로 줄어듦

### 4.3 목표 카드 UI (가로 스와이프)
- 목표가 여러 개일 때 **좌우 스와이프**로 전환
- dot indicator로 현재 카드 위치 표시
- 각 카드 구성:
  - 미션명 + D-day
  - 오늘 할당량 (큰 숫자)
  - **프로그레스 바** (전체 대비 완료 비율)
  - 3개 선택 버튼

### 4.4 미루기 캘린더
- 세번째 레퍼런스(수면 앱)처럼 **날짜별 캐릭터 표정**으로 표시
- 했어요: 😊 / 조금 했어요: 😐 / 미뤘어요: 😨 / 안 함: 빈칸
- 월별 보기, 한눈에 미루기 패턴 파악 가능
- 홈 하단에 간략히 표시 (이번 주만), 탭하면 전체 캘린더

### 4.5 빈 화면 (목표 없을 때)
- 원기왕성한 캐릭터 + "나랑 같이 벼락치기 해볼래?" 문구
- [목표 만들기] CTA 버튼

### 4.4 선택지 (목표별)
1. **"했어요 ✓"** — 오늘 할당량 완료 → 캐릭터 축하 애니메이션
2. **"조금 했어요"** — **슬라이더**로 진행량 입력 (0~할당량) → 나머지 재계산 → 팩트 폭행
3. **"미룰래요 ✗"** — 오늘 전체 미루기 → 팩트 폭행

### 4.5 미루기 제한
- 하루 **제한 없음** (여러 번 미룰 수 있음)

## 5. 핵심 기능 C — 팩트 폭행 (전체화면 경고)

### 5.1 연출
- 미루기 선택 시 **전체화면**으로 전환
- 피폐해진 캐릭터 (상태별 이미지)
- 재계산된 숫자 (큰 폰트 강조)
- 경고 문구 (단계별 톤 변화)
- [공유하기] + [돌아가기] 버튼

### 5.2 경고 문구 톤 (단계별 변화)

| 미루기 횟수 | 톤 | 예시 |
|-------------|-----|------|
| 1회 | 친절 | "괜찮아, 내일 8p씩이면 돼!" |
| 2회 | 살짝 걱정 | "오늘도 미뤘네? 내일 12p야..." |
| 3~4회 | 직설적 | "야... 23p인데 괜찮아?" |
| 5회+ | 포기 | "...나도 모르겠다" |

### 5.3 캐릭터 상태 + 배경 변화

| 미루기 횟수 | 캐릭터 상태 | 배경 톤 |
|-------------|------------|---------|
| 0회 | 원기왕성 😄 | 밝은 화이트 |
| 1~2회 | 살짝 불안 😐 | 살짝 회색끼 |
| 3~4회 | 땀 흘리는 중 😨 | 어두운 톤 |
| 5회+ | 버텨요.. 💀 | 거의 검정 |

### 5.4 전면 광고 타이밍
- 팩트 폭행 화면에서 **"돌아가기" 버튼** 누를 때 전면 광고 표시
- 팩트 폭행 자체는 방해 없이 보여줌

## 10. 화면 흐름 (IA)

### 10.1 화면 구성
- **원페이지** (탭 없음)
- 상단 우측: [⚙] 설정 + [+] 목표 추가 FAB

### 10.2 화면 목록
1. 토스 로그인
2. 별명 설정
3. **홈** — 캐릭터(+요약 수치) + 목표 카드 리스트
4. **목표 추가 바텀시트** — 미션명/마감일/분량/단위 입력
5. **진행량 입력 바텀시트** — 슬라이더로 "조금 했어요" 입력
6. **팩트 폭행 전체화면** — 피폐 캐릭터 + 경고 문구 + 공유 버튼
7. **완료 축하** — 캐릭터 축하 + 애니메이션
8. **목표 종료 결과** — 마감일 도래 시 결과 카드 + 공유
9. **설정** — 프로필, 목표 관리, 알림 설정, 이용약관, 탈퇴

### 10.3 주요 플로우
```
[로그인] → [별명 설정] → [홈]
                            │
                            ├── [+] → [목표 추가 바텀시트]
                            │
                            ├── [했어요 ✓] → 축하 애니메이션
                            │
                            ├── [조금 했어요] → [슬라이더 바텀시트] → 팩트 폭행 → [공유]
                            │                                              ↓
                            │                                     [돌아가기] → 전면 광고
                            │
                            ├── [미룰래요 ✗] → 팩트 폭행 → [공유]
                            │                       ↓
                            │              [돌아가기] → 전면 광고
                            │
                            └── [⚙] → [설정] → 프로필/목표관리/알림/약관/탈퇴
```
